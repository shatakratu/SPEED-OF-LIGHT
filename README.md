# 🌍 Speed of Light Simulation



## 🔭 Physical Scale & Accuracy


| Constant | Physical Value |
| :--- | :--- |
| **Scale Factor** | 1:1,000,000 ($10^{-6}$) |
| **Speed of Light ($c$)** | $299,792,458 \text{ m/s}$ |
| **Average Distance** | $384,400,000 \text{ m}$ |
| **Earth Radius** | $6,371,000 \text{ m}$ |
| **Moon Radius** | $1,737,400 \text{ m}$ |
| **Time of Flight** | $1.282 \text{ seconds}$ |



---

## 🛠️ Prerequisites

*   **Python 3.8+**
*   **uv** (python package manager)

---

## 🚀 Quick Start

### 1. Clone & Navigate to Project
```bash
git clone https://github.com/shatakratu/SPEED-OF-LIGHT.git
cd SPEED-OF-LIGHT
```

### 2. Install `uv` (if not already installed)
* **Linux / macOS:**
  ```bash
  curl -LsSf https://astral.sh/uv/install.sh | sh
  ```
* **Windows (PowerShell):**
  ```powershell
  powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
  ```

### 3. Initialize & Run Environment

```bash
# Initialize
uv init

# Add Pygame dependency to the project
uv add pygame

# Run the simulation script
uv run main.py
```
